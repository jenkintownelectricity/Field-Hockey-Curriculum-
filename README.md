# Field Hockey Curriculum SaaS

This repository contains the FieldHockeyKernel™ based youth field hockey coaching system.

The project is built using a kernel-first architecture where drills, programs, surfaces, materials, pedagogy, and emotional learning cues are structured data.

The system generates:

• coaching workbooks  
• SVG field diagrams  
• animated drill storyboards  
• YouTube training scripts  
• 6-week youth development programs  
• mobile-ready SaaS coaching platform  

See `.claude/DIRECTIVE.md` for the full build directive.
